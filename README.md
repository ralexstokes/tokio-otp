# tokio-supervisor

Structured task supervision for tokio.
Inspired by Erlang/OTP-style supervision trees.

## Getting started

Check the Rust docs for more information, e.g. `just doc`.

## Development

Use the flake for both local tooling and CI:

```sh
nix develop
just ci
```
